# 快速测试指南

## 已实现的新功能（本次更新）

### ✅ 拖拽功能
1. **从组件库拖拽到画布**
   - 打开应用，创建新项目
   - 从左侧组件库拖拽任意组件到画布
   - 画布会显示拖拽悬停效果（蓝色虚线边框）
   - 释放鼠标后组件会添加到画布

2. **组件选中和高亮**
   - 点击画布上的组件可以选中
   - 选中的组件有蓝色实线边框
   - 悬停的组件有蓝色虚线边框
   - 右侧属性面板会显示选中组件的配置

### ✅ 自动保存
- 修改组件后自动触发保存（500ms 防抖）
- 项目路径已设置时会自动保存
- 支持手动保存（Ctrl/Cmd + S）

### ✅ 组件库扩展
新增组件：
- **容器组件**: Grid容器、卡片
- **基础组件**: 标题、分割线、链接
- **表单组件**: 下拉框、复选框、单选框、开关

### ✅ 组件渲染优化
- 修复了子组件递归渲染逻辑
- 容器组件自动添加默认样式（边框、内边距）
- Flex 容器自动应用 flex 布局
- 正确处理组件文本内容

## 测试步骤

### 1. 启动应用
```bash
# 确保开发服务器正在运行
npm run dev

# 在浏览器访问
http://localhost:5173
```

### 2. 创建项目
1. 点击"新建项目"
2. 输入项目名称（如：我的测试项目）
3. 选择 UI 库（默认 Element Plus）
4. 选择项目保存位置
5. 点击"创建"

### 3. 测试拖拽功能
1. **添加容器**：
   - 从左侧"容器组件"拖拽"Flex容器"到画布
   - 观察画布上出现带边框的容器

2. **添加按钮**：
   - 从"基础组件"拖拽"按钮"到画布
   - 点击按钮，查看是否被选中（蓝色边框）
   - 右侧属性面板显示按钮属性

3. **修改属性**：
   - 选中按钮
   - 在右侧"属性"标签页修改按钮文本
   - 切换到"样式"标签页修改颜色、尺寸

4. **添加更多组件**：
   - 继续添加文本、图片、输入框等组件
   - 尝试添加"Card卡片"等容器组件

### 4. 测试页面管理
1. 点击左侧"页面"标签
2. 点击"新建页面"
3. 输入页面名称和路由
4. 在不同页面间切换

### 5. 测试撤销/重做
1. 添加几个组件
2. 按 Ctrl/Cmd + Z 撤销
3. 按 Ctrl/Cmd + Shift + Z 重做

### 6. 测试保存
1. 修改组件后等待 0.5 秒（自动保存）
2. 或按 Ctrl/Cmd + S 手动保存
3. 查看顶部工具栏的保存状态

## 已知限制

### 当前版本不支持：
- ❌ 组件拖拽移动位置
- ❌ 组件尺寸调整
- ❌ 组件复制/粘贴
- ❌ 组件删除（需在代码中手动实现）
- ❌ 容器内拖放子组件
- ❌ 代码模式
- ❌ 导出功能

### 下一步开发：
1. **删除组件** - 添加删除按钮或 Delete 键支持
2. **容器内拖放** - 支持将组件拖入容器内部
3. **组件树视图** - 显示完整的组件层级
4. **代码模式** - Monaco Editor 集成
5. **导出功能** - 生成可运行的 Vue 项目

## 调试技巧

### 查看组件状态
在浏览器控制台输入：
```javascript
// 查看当前页面组件树
window.__CANVAS_STORE__ = useCanvasStore()
console.log(window.__CANVAS_STORE__.currentComponents)

// 查看选中的组件
window.__EDITOR_STORE__ = useEditorStore()
console.log(window.__EDITOR_STORE__.selectedComponentId)
```

### 查看 Pinia 状态
安装 Vue Devtools 浏览器扩展，可以实时查看所有 store 的状态
